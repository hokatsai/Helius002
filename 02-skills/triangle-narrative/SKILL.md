# 三角结构叙事 Skill（Triangle Narrative）

## 功能说明
基于「三角结构」内容创作方法论，帮助生成认知升级、金钱心理学、个人成长类深度视频内容。涵盖三角脚本结构、14镜头模板、8大内容支柱、爆款标题公式和变现路径。

> 📝 **方法论来源**：本风格模板提炼自 MoneyXYZ（一口新飯）频道的创作体系。

## 核心能力
- **三角脚本引擎**：Hook → Story → Logic → Emotion → CTA 五步结构
- **14镜头视频模板**：可套用的标准化视频拍摄结构
- **爆款标题生成**：基于频道验证的标题公式库
- **8大内容支柱**：三角结构体系的核心主题板块
- **变现路径分析**：从免费内容到会员制的价值阶梯

## 输入/输出规范

### 输入
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| topic | string | 是 | 要创作的主题（如"延迟满足"、"反脆弱"） |
| format | string | 否 | 输出格式：script（完整脚本）/ outline（大纲）/ title（标题） |
| pillar | string | 否 | 指定内容支柱（见下方列表） |

### 输出
| 参数 | 类型 | 说明 |
|------|------|------|
| script | string | 完整的视频脚本（三角结构） |
| outline | string | 视频大纲（Hook/Story/Logic/Emotion/CTA） |
| title | string | 3-5个备选标题 |

## 二、三角脚本结构（V2 脚本引擎）

MoneyXYZ 的视频脚本遵循 **三角结构**，每个模块有明确功能：

### 🔺 Hook（钩子）— 反直觉开场
**功能**：3秒内抓住注意力，推翻观众已有认知

**公式**：
```
"你以为 XXX，但真相是 XXX"
"大多数人在 XXX 上犯了同一个错误"
"我曾经相信 XXX，直到我发现 XXX"
```

**示例**：
> "你以为省钱是财富自由的钥匙？但纳瓦尔告诉你，赚钱比省钱重要100倍。"

---

### 📖 Story（故事）— 权威数据 + 书籍引用
**功能**：建立信任感，用权威背书填充内容

**元素**：
- 权威数据（具体数字 > 模糊描述）
- 书籍引用（至少2本，用「书名+核心观点」格式）
- 人物案例（最好是自己的经历或身边故事）

**公式**：
```
引用1（数据/书/名人）→ 解释其意义 → 引用2 → 总结共通点
```

---

### 🧠 Logic（逻辑）— Point 1/2/3 模块化
**功能**：提供可操作的洞察，分3个分论点

**格式**：
```
Point 1：[反直觉观点] + [解释] + [例子]
Point 2：[反直觉观点] + [解释] + [例子]
Point 3：[反直觉观点] + [解释] + [例子]
```

**原则**：
- 每个 Point 独立可记，利于观众截图传播
- 用"因为...所以..."逻辑链串联
- 避免信息过载，每 Point 不超过3句话

---

### ❤️ Emotion（情感）— 个人坦白/反转
**功能**：制造情感共鸣，完成从"知识"到"信念"的转变

**公式**：
```
[个人失败经历] 或 [极端假设/反转]
→ 袒露脆弱或展示黑暗面
→ 与观众建立深层连接
```

**示例**：
> "我曾经也以为只要努力工作就能致富，结果30岁还背着学生贷款。"

---

### ✅ CTA（行动）— 极简微行动
**功能**：给出1个观众今天就能做的小行动

**原则**：极简、具体、可立刻执行

**公式**：
```
"今天，只需要做一件事：[具体微行动]"
```

**示例**：
> "今天开始，读任何书的时候，问自己一个问题：这本书教我的哪件事，能让我多赚1000块？"

## 四、爆款标题公式

### 公式A：颠覆型
```
"[普遍认知]是最大的谎言"
"为什么[共识]其实是错的"
"[数字]个关于[主题]的误区"
```

### 公式B：数字型
```
"[数字]个[领域]真相（大多数人不知道）"
"[数字]年内不会告诉你的[主题]"
"[数字]步教你[结果]"
```

### 公式C：好奇型
```
"为什么[受众]的[愿望]永远无法实现"
"[反问]？（大多数人都答错了）"
"如果[假设]，你会[结果]吗"
```

### 公式D：对比型
```
"穷人的[习惯] vs 富人的[习惯]"
"[普通人]和[成功人士]的[差异]"
"一个月后，你和现在的你的[对比]"
```

### 标题库示例
| 类型 | 标题 |
|------|------|
| 颠覆型 | "延迟满足是最大的谎言" |
| 数字型 | "3个纳瓦尔不会告诉你的致富秘密" |
| 好奇型 | "为什么你努力了10年还是穷？" |
| 对比型 | "有钱人和穷人的最大差异，不是努力" |
| 数字型 | "5分钟讲清楚什么是反脆弱" |
| 颠覆型 | "读书无用论？99%的人都读错了书" |

## 五、14镜头视频模板

MoneyXYZ 的视频结构可拆解为 **14个镜头**，每个镜头有固定时长和功能：

| 镜头# | 时长 | 类型 | 内容要点 |
|-------|------|------|---------|
| 1 | 0-3s | **开场Hook** | 反直觉陈述，推翻共识 |
| 2 | 3-10s | **问题引入** | 抛出观众痛点/问题 |
| 3 | 10-20s | **过渡** | 预告接下来要讲的内容 |
| 4 | 20-40s | **故事1** | 数据/书籍/权威引用（Point 1背景） |
| 5 | 40-60s | **Point 1** | 第一个核心观点 + 例子 |
| 6 | 60-80s | **过渡** | 连接词："但这还不是最重要的..." |
| 7 | 80-100s | **故事2** | 第二个数据/案例（Point 2背景） |
| 8 | 100-120s | **Point 2** | 第二个核心观点 + 例子 |
| 9 | 120-130s | **过渡** | "最重要的是..." |
| 10 | 130-150s | **故事3** | 个人经历或极端案例（Point 3背景） |
| 11 | 150-170s | **Point 3** | 第三个核心观点 + 例子 |
| 12 | 170-180s | **Emotion** | 个人坦白/反转/情感共鸣 |
| 13 | 180-190s | **总结** | 3个Point一句话回顾 |
| 14 | 190-200s | **CTA** | 极简微行动 + 关注引导 |

**总时长参考**：约3-5分钟（Shorts版）/ 8-15分钟（长视频版）

## 六、变现路径

MoneyXYZ 的变现遵循 **价值阶梯** 模式：

```
免费内容（YouTube）
    ↓
低价产品 / 资源包（$9-29）
    ↓
会员网站 onenewbite.com（订阅制）
    ↓
高价课程 / 训练营（$299-999）
    ↓
私教 / 1v1 服务（$2000+）
```

**关键原则**：
- 每个层级都需要前一层级的信任积累
- 内容即销售，不要在免费内容里硬推销
- 变现产品要解决观众的"下一个问题"

## 七、创作工作流

### 日/周创作节奏
| 时间 | 动作 |
|------|------|
| 每天30分钟 | 阅读（书/Newsletter/论文）→ 提取1个核心观点 |
| 每周1个主题 | 从观点库选1个主题 |
| 写作2小时 | 按三角结构写完整脚本 |
| 拍摄1小时 | 按14镜头模板录制 |
| 剪辑30分钟 | 字幕 + 关键帧 + B-roll |

### 选题来源优先级
1. **评论区问题**：观众问了什么？
2. **热门视频拆解**：同类爆款但换角度
3. **书籍核心观点**：哪本书被引用最多但没讲透？
4. **个人经历**：你自己踩过的坑
