# 坚持5点起床一年，我搞懂了早起改变人生的秘密

---
## 💡 創作者思路與借鑒指南 (Creator's Insights)
- **核心破局點 (The Reframing)**: 放棄講述老生常談的「靠意志力堅持」，轉而提出「無利不起早」的概念。把「早起」這項痛苦的行為，包裝成「自私地佔用最高效時間」的超高回報投資，徹底釋放觀眾起不來的內疚感。
- **神級視覺錨點 (Visual Anchors)**: 發明了「大腦＝罐子，每天抱著空罐子跑 100 米」的絕佳比喻。觀眾甚至不需要看畫面，憑藉文本就能牢牢記住「不要把最好、最空的罐子送給老闆」這個視覺意象。
- **結構優勢 (The Rule of 3)**: 在給出具體 `5-am project` 建議時，使用黃金三分法（1.特別需要做的、2.特別想做的、3.特別不同的事），極大地降低了觀眾的認知與吸收負擔。
- **無縫軟廣植入 (Value-First Sponsor)**: 把 Lingoda 業配當作自己執行 `5-am project` (學西班牙語) 的「真實案例」，將廣告化為知識點，實現 0 違和感的商業轉化。

## 🎬 腳本文本與結構標註 (Annotated Script)

### 📌 [黃金開場 Hook] 數據結果前置，懸念拉滿
> *分析：不講廢話，用真實數字（堅持了25天➜現在滿一年）直接破題，建立強大的結果導向權威。*

2021年8月 21号我在YouTube社区发了一条状态，说我在读了那本著名的5 AM CLUB 之后，已经坚持5点起床25天了。我当时之所以要“早起”就是想弄清楚这个”早起”到底是不是像那本书里说的那么神奇，能够让你效率倍增，甚至彻底改变人生。今天是2023年的1月5号，我坚持早起一年多的时间了，我想我终于搞懂了早起的奥义。

那在这个视频里我就结合自己的真实体会，聊一聊我们对早起的误解以及如何做才能让”早起”真正帮你改变人生。那视频的最后我也会分享一些帮助你早起的实用技巧。现在是早上5点15，在这次尝试之前，我其实有过两次失败的经历。那两次都比较激进，是4点起床。但是每次都是坚持不到一礼拜就放弃了。而我这次能够早起成功呢，最主要是我当时经历了一个偶然。

### 📌 [破除迷思] 摧毀意志力神話與錯誤因果
> *分析：指出成功人士早起不是靠意志力長志氣（打碎舊認知），消除觀眾焦慮。*

而这次偶然也让我找发现了我之前失败的真正的原因。你看啊，我们经常听到很多名人早起的例子，比如富兰克林早上5点起床，和迪斯尼的 Robert Iger都是四点半起床等等。这些例子在我们潜意识里建立了一种因果关系。那就是他们的早起导致了他们的成功。但是问题是，即使是早起和成功之间存在某种因果关系，它显然也不是立即发生的，当这种好处不立竿见影的时候，光靠意志力是很难坚持下去的。

我之前每次失败后都会想起小时候老师批评我们的那句话”常立志不如立长志”。对，我就是那个常立志的失败者。然而我这一年多早起的经历让我获得了一个新的角度，有没有可能就是那些成功人士的早起并不是所谓的”立长志”,它更可能是一种“无利不起早”。早起的”好处“对他们来说可能更加立竿见影。

### 📌 [核心解藥] 提出全新手段：the 5-am project
> *分析：明確指出兩個核心奧義，將早起從「目的」變為「手段」，帶出專屬框架法則。*

那我把这种好处归结为两点，我认为它们才是早起的真正奥义。第一，我们应该focus在早起要做的事情上，而不是早起本身。第二，深刻懂得”早起“实际上是和时间进行的一种”田忌赛马“的游戏。

那我们先从第一点讲起。我一年前的这次尝试之所以能成功，是因为我当时有一个迫切需要解决的问题。那就是学习如何做动画。我这个是一个偏知识类的频道，很多视频都需要使用动画效果来呈现一些数据和原理。早期的时候我跟很多人一样从网上找现成的模板，在刚开始的时候他们确实能帮我解决问题，但是随着视频做的越来越多，越来越难，这些模板很难满足我特定的需求，非常浪费时间。我急切需要从根本上解决这个问题，学会做动画。

于是我就把每天早上的6点到8点这段时间全部用来学做动画。这就是我当时的5-am project。这件事儿让我很快尝到了甜头。我每天学到的东西都能立即用到我的视频制作当中，开始帮我节省时间。然后我就突然明白了，我之前每次尝试早起失败是因为我在做一些别人告诉我该做的事情，比如什么冥想, 阅读, 运动或者去学一个我自己并不真正需要的软件和工具。这里不要误会我的意思，我不是说这些事情不好，我是说这些东西不是我需要的。

我当时更想解决一个急迫的问题，学做动画能给我提供更大和更立竿见影的好处。这就是我的”无利不起早”。我到现在都还记着刚开始那几天每次学完动画之后看看表也就早上8点多的时间，对很多人来说新的一天也才刚刚开始，而我已经学了很多实用的东西，特别有成就感。

### 📌 [具體指引] 三分法教學尋找自己的專屬目標
> *分析：使用 1、2、3 法則列舉專案種類，這是最標準的知識型博主降低吸收負擔的結構。*

所以我认为，想要真正做到早起，你需要一个明确的5-am project,这个Project可以有三种可能。那第一类5-am project 呢可以是你特别需要的做的事情所谓的”需要”就是能帮你解决问题的事情。我学动画就属于这一类。那每个人的需要是不同的。比如前几天我一个学生说他要把自己的摄影作品打印出来，但是不会弄。那学习照片的打印就可以是他的5-am project。再比如你需要考取一个证书，那么准备考试就可以是你的5-am project等等。

第二类5-am project可以是你特别想要做的事情。比如，我平时读非小说类的书籍很多，但是我一直想读一些小说类的书籍。所以我有一阵子的5-am project,就是去读指环王的三部曲。你看我这里有原版的三部曲还有一些画册那我当前的5-am project 就是学习西班牙语。

### 📌 [回歸正題] 提出第三法則，拔高維度
> *分析：結束廣告後迅速給予額外乾貨補償，引入 Google 20% 法則，將眼界拔高。*

OK，我们继续。除了自己需要和想要做的事情外，第三类5-am project 可以是一件”特别不同的事情”。所谓的特别不同，是指和你每天的学习工作不是特别相关的事情。它可以是一个你从来没想过要做的事情。比如你是个软件工程师，你可以早上做一些和艺术相关的事情，研究一下艺术史啊，学一学如何写歌儿作曲啊，学个园艺室内装修设计等等。当然这个特别不同的事情和特别想做的事情可能是相同的。比如我学西班牙语。

我认为做不同的事很重要，因为它可能给你提供新的机会和可能，也能帮你提高创造力。这就好像Google非常有名的20% 法则,那就是Google鼓励员工花费20%的时间用在和自己常规工作不直接相关的项目上。据说Google很多重要的产品都诞生于这些所谓“不相关”的项目中。那5-am project就相当于我们自己的20%法则。

### 📌 [神級視覺錨點] 大腦空罐子與田忌賽馬
> *分析：創造了全片流傳度最高、最具影像感的比喻（空罐子理論），建立「自私護時」正當性。*

那说到这里可能有人会说，为什么非得是早上起来做这些事儿呢？难道我晚上熬夜不可以吗? OK, 这个问题的核心其实就是早起的奥义的第二点，那就是早起实际上是和时间做“田忌赛马”的游戏。

想要理解这个游戏，你需要知道大脑和时间的两个特点。第一，我们每天的生活就像是抱着一个罐子在做100米冲刺跑。我们的大脑就是这个罐子。早上起来的时候它是空的，也是我们最有能量的起跑阶段。随着一天的推进，我们大脑接受和处理的信息和噪音会越来越多，这个罐子就会越来越满，我们越跑越慢。当我们最后冲刺的时候也是我们最疲惫的时候，这时候我们就需要去睡觉，把这个罐子清空，第二天继续起跑。

第二，我们大脑这个罐子里装的东西很多时候并不属于我们自己。如果你起床之后就是吃饭上班，那么你就把大脑最有能量的时间给了同事，老板，客户，商店，餐馆，等等。这就好像你抱着这个罐子刚起跑，别人就开始往里装他们的东西,占据它的空间。当你快到终点的时候，你的罐子已经快满了。如果这时候你决定做自己的Project，要么这个罐子没空间了，要么你已经抱不动这个罐子了。实际上，对于我们大多数人来说，从下班到晚上睡觉前是垃圾时间。我们大多数人在这段时间里效率很低，只能做一些无脑的机械的事情。比如刷手机，看电视，和别人吃饭等等。

更糟糕的是，由于手机和社交网络的发达，我们很多人会把这段垃圾时间无限延长，经常刷手机刷到后半夜。这其实也是一种FOMO，不看手机的时候就好像错过了这个世界一样。所以总结下来，我们一天中拥有最好精力和最高效率的时间都被别人占据了。很多人其实没有属于自己的时间，更不要提什么自我发展。早起做自己的5-am project就是和时间做田忌赛马的游戏。我们把7点或8点起床提前到5点，从5点到8点就是你做personal project的时间。

你现在抱着一个空罐子起跑，而且往罐子里装都是自己的东西。早起的意义就在于我们很”自私”的把大脑最宝贵的空间和效率主动留给了自己。这段时间是别人想侵犯都无法侵犯的。因为现在整个世界都还没有醒过来，这时候没有工作，同事，甚至家人的打扰，轮不到别人去填满你的罐子。我给大家看看早上5点多外面是什么样子啊一个人也没有然后天上还有月亮所以，那些成功人士早起的原因可能不是他们意志力有多高，而是早起这一段时间是唯一属于他们自己的时间。他们为了自己在和时间做田忌赛马的游戏。

### 📌 [結尾收束] 解決心理疑慮與社群呼籲
> *分析：告訴觀眾「不用每天早起，這不是坐牢」，大幅降低觀眾執行壓力，完美引導點贊留言。*

那这时候有人可能会问了，如果早上5点起床，我怎么去保证七八个小时的睡眠呢？这个答案很简单，那就是9点或者10点上床睡觉。因为从我们下班到睡觉这段时间本来就是被手机或者别人占领的，是大多数人本来就会浪费掉的垃圾时间。所以把一部分垃圾时间变成睡眠和5-am project的时间是非常合算的。这也是在和时间进行田忌赛马。

实际上当你把这个5-am project 定义好之后你起床就不会那么的难了不过这里必须要强调的一点就是你没有必要逼迫自己后半生的每一天都要早起别忘了，我们起来是为了做这些personal project的。当一个project结束之后呢你可以中断一段时间，不需要感到内心有内疚。而且，我们的生活也不允许我们每天早起。比如我今年会去欧洲旅行一个多月，横跨好几个时区，那我是不可能每天早起做5-am project。

所以，听到这里你明白了早起的奥义了吗？如果你想做自己特别需要，特别想要，或者特别不同的事情去提升自己，对于大多数人来说，早起做自己的5-am project可能是你最务实也是唯一的机会。

那这个早起当然也是有技巧的那像我每天早上起来之后就是先会咕咚咕咚喝一大杯水然后就出去散步十几分钟左右那如果天气太冷的话我会站到我家后院让冷空气把我给吹醒那其实你也可以去做那种稍微剧烈一点的运动让自己身上出汗但是我不是特别喜欢啊那实际上就是你在坚持了差不多两周左右你的生物钟就自然的把你叫醒就没那么困你像我今天早上就是四点五十就醒了再怎么也睡不着了那想早起的第二点就是为了防止我们看手机啊头天晚上睡觉之前就可以把手机放到客厅里那早上你可以拿那个闹钟或者说你像我拿这个Apple Watch来把自己呃给叫醒因为这个东西呢就是到点之后它会在你手腕上玩命的震但是也不会去吵到别人这是挺好的方法。

那我也知道就是很多人想提前睡九点十点睡就是怎么睡也睡不着而且第二天醒来之后还会不舒服那这个其实有科学根据的就是研究发现如果你这个人超过就是打破你平时的入睡时间超过一个小时那无论你晚上睡多久你第二天怎么去补觉你都会感到非常的累所以呢如果你想调节睡眠就是每次以半个小时为基准来提前半个小时睡觉然后提前半个小时起床直到达到你的这个目标的起床时间为止早起其实不是成功的原因，你的5-am project才是把你和别人区分开的重要因素。
