# Analysis Framework

Use this structure for `analysis_report.md`.

## 1. Basic info
- Title
- Creator
- Platform
- URL
- Confidence level (high/medium/low)

## 2. Core topic
Answer:
- What is the video really about?
- What anxiety, desire, or curiosity is it addressing?

## 3. Click logic
Break down:
- contrast / conflict
- promised outcome
- curiosity gap
- identity resonance

## 4. Structure breakdown
Map these when possible:
- Hook
- Setup / framing
- Reframe / counterintuitive turn
- Supporting logic or examples
- Actionable takeaway
- CTA / close

## 5. Persuasion and retention
Look for:
- emotional turns
- authority signals
- story devices
- pacing changes
- pattern interrupts

## 6. Extractables
Produce:
- Hook candidates
- 3-10 key lines
- CTA wording
- reusable framework
- thumbnail/title lessons if visible

## 7. Style mapping
Compare against known creator styles:
- MoneyXYZ = authority + data/book references + structured logic
- Dan Koe = identity + philosophy + creator business systems
- 認知便利店M = cognitive reframing + commercial psychology + AI visual packaging

## 8. Reusable value for Helius
Always end with:
- What to learn
- What to copy
- What not to copy blindly
- 3 new content angles Helius can create from this
